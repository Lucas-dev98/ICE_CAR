# ICE CAR - Site Institucional Automotivo 🚗❄️

Site institucional moderno e responsivo para empresa de ar-condicionado automotivo, desenvolvido com **React** e **Vite**.

## 📁 Estrutura do Projeto

```
src/
├── components/
│   ├── Header.jsx              # Navbar com menu mobile
│   ├── Header.css
│   ├── Hero.jsx                # Seção hero com partículas animadas
│   ├── Hero.css
│   ├── Sobre.jsx               # Sobre a empresa
│   ├── Sobre.css
│   ├── Servicos.jsx            # Grid de 6 serviços
│   ├── Servicos.css
│   ├── Stats.jsx               # Contadores animados
│   ├── Stats.css
│   ├── Diferenciais.jsx        # 6 cards de diferenciais
│   ├── Diferenciais.css
│   ├── Depoimentos.jsx         # 3 testimonial cards
│   ├── Depoimentos.css
│   ├── Contato.jsx             # Formulário + informações
│   ├── Contato.css
│   ├── Footer.jsx              # Footer com links
│   ├── Footer.css
│   ├── FloatingButtons.jsx     # Back-to-top + WhatsApp
│   └── FloatingButtons.css
├── App.jsx                     # Componente raiz
├── App.css                     # Estilos globais
├── main.jsx                    # Entry point
└── index.css                   # Estilos base

```

## 🚀 Como Rodar

```bash
cd /home/lucasbastos/ICE_CAR
npm install
npm run dev
```

Servidor inicia em `http://localhost:5173`

## ✨ Recursos

- ✅ Componentizado (React)
- ✅ Responsivo (Mobile-first)
- ✅ Animações suaves
- ✅ Scroll reveal com IntersectionObserver
- ✅ Formulário com validação
- ✅ Máscara de telefone
- ✅ Contadores animados
- ✅ Menu mobile com hamburger
- ✅ Smooth scroll
- ✅ WhatsApp integration

## 🧩 Componentes

1. **Header** - Navegação fixa + menu mobile
2. **Hero** - Seção principal com partículas
3. **Sobre** - Apresentação com 21 anos
4. **Serviços** - Grid de 6 cards
5. **Stats** - 4 contadores animados
6. **Diferenciais** - 6 cards principais
7. **Depoimentos** - 3 testimonios
8. **Contato** - Formulário + info
9. **Footer** - Links e copyright
10. **FloatingButtons** - Back-to-top + WhatsApp

## 📱 Responsividade

- Desktop: 1200px+
- Tablet: 768px - 1199px
- Mobile: até 767px
